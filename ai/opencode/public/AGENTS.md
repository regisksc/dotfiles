# context-mode + Skills

> **Note:** Full configuration is in `~/.config/opencode/CLAUDE.md` and `~/.claude/CLAUDE.md`

## Quick Reference

1. **SKILLS FIRST** ‚Äî Use specialized skills when available (humanizer, security-review, flutter-*, etc.)
2. **context-mode for data** ‚Äî Research, analysis, fetching, processing

## Skill Examples

- Writing/editing ‚Üí `humanizer`
- Security ‚Üí `security-review`
- Testing ‚Üí `tdd-workflow`, `python-testing`
- Flutter ‚Üí `flutter-*` skills
- UI/UX ‚Üí `ui-ux-pro-max`
- Research ‚Üí `smart-explore`

## Commands

- `/ctx stats` ‚Äî Session statistics
- `/ctx doctor` ‚Äî Diagnostics
- `/ctx upgrade` ‚Äî Upgrade context-mode
